# Analyzing GVB particles

## Getting started

### Prerequisites

- [Python](https://www.python.org/downloads/) version 3.14 or higher

### Download code

1. Download or clone the GitHub repository
   ```bash
   git clone git@github.com:vaioic/henderson-lab-brain-gvb.git
   cd henderson-lab-brain-gvb.git
   ```
### Python setup

If running the code for the first time, you will need to create a Python virtual environment and install the necessary packages. 

1. Open a terminal and navigate to the directory where you unzipped the files.

2. Create a python virtual environment
   ```bash
   python -m venv venv
   ```

3. Activate the virtual environment
   Windows:
   ```bash
   .\venv\Scripts\activate
   ```
   
   Linux:
   ```bash
   source venv/scripts/activate
   ```

4. Install the dependencies using Pip
   ```bash
   python -m pip install -r .\requirements.txt
   ```
